這是一個幫助 WordPress 寫作者更輕鬆自動化內容維護的小工具包。支援 [year] 和 [postlink] 兩種實用短代碼。

== Description ==

**文章寫作好幫手** 由老K＆BRAND 自媒體課程製作，協助你在 WordPress 寫作時減少重複工作。

支援：

1. `[year]` → 自動顯示當前年份，可用於文章內文、標題、SEO 標題（Rank Math）
2. `[postlink]文章連結[/postlink]` → 自動產生該文章的標題超連結，未來標題改了也不怕！

== Installation ==

1. 下載並安裝此外掛（可從 WordPress 後台 → 外掛 → 上傳外掛）
2. 啟用後，即可在文章中使用 `[year]` 和 `[postlink]` 短代碼

== 使用方式 ==

- 顯示今年年份：
  `[year]`

- 自動連結到文章（會顯示標題）：
  `[postlink]https://你的網站.com/文章連結[/postlink]`

- 也可以只輸入子目錄（WordPress 網站中的相對路徑）更簡潔：
  `[postlink]/post-name[/postlink]`  
  外掛會自動補上目前網站網域，變成完整網址並抓出文章標題。
